# 🎮 PoppyPlaytime FPS Unlocker

A lightweight utility designed to improve performance and unlock higher frame rates in **Poppy Playtime** on supported Windows systems.

> ⚡ Increase smoothness, reduce input delay, and remove default FPS caps for a better gameplay experience.

---

## ✨ Features

- 🔓 Unlocks FPS beyond the default in-game limit
- ⚙️ Automatic config detection
- 🚀 Performance mode for low-end PCs
- 🖥️ Supports 60 / 120 / 144 / 240 FPS targets
- 🛡️ Safe memory patching (no game file edits)
- 📉 Lower input latency
- 🎯 One-click apply & restore defaults

---

## 📋 Requirements

- Windows 10 / 11 (64-bit)
- Installed copy of **Poppy Playtime**
- DirectX 11 compatible GPU
- .NET Runtime 6.0+

---

## 📦 Installation

1. Download latest release from the Releases page.
2. Extract the `.zip` file.
3. Run `PoppyFPSUnlocker.exe` as Administrator.
4. Launch the game.
5. Select your target FPS and click **Apply**.

---

## 🛠️ How It Works

The tool automatically detects Unreal Engine frame cap settings used by Poppy Playtime and temporarily overrides them during runtime.

It can also optimize:

- VSync behavior
- Frame pacing
- Input polling rate
- Shader cache settings
